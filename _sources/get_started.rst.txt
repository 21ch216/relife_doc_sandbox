Get started
===========

.. role:: python(code)
   :language: python

.. rubric:: Objects import

Launch a Python interpreter. All ReLife objects can be imported from ``relife``. Thus, all you have to do is
``from relife import <an-object>`` (remember that in Python everything is an object, even class). For instance, if you
want to import Weibull constructor, just write :

.. code-block:: console

    from relife import Weibull


.. note::

    One may be tempted to write ``from relife import *``. Of course you can. We prefer the above command as it is more
    explicit
